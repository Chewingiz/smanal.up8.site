# smanal.up8.site
Ce projet est un site web que j'ai développé pendant ma première année d'études (L1).

## Objectifs
L'objectif principal de ce projet était de concevoir et de développer un site web statique pour présenter un sujet qui m'intéresse. J'ai choisi de mettre en avant mes réalisations artistiques, tels que des dessins et des tableaux. 
En développant ce site, j'ai également cherché à améliorer mes compétences en HTML et CSS.
## Fonctionnalités
* Présentation du sujet : Le site web comprend une page d'accueil qui présente le thème central de mon travail artistique.
* Galerie d'images : J'ai créé une galerie d'images où je présente mes dessins et mes tableaux.
* Conception personnalisée : J'ai réalisé moi-même toutes les icônes et les éléments visuels du site pour donner une touche personnelle à la présentation. Voici quelques exemples d'icônes que j'ai créées :
